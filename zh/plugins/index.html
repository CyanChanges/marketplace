    基于 photo-swipe 的图片预览插件

    返回顶部按钮

    @vuepress/plugin-medium-zoom

    将 medium-zoom 集成到VuePress。

    vuepress-plugin-lightgallery

    基于 lightgallery 的图片预览插件

    Markdown

    @vuepress/plugin-container

    在你的 VuePress 站点中注册 markdown 自定义容器

    vuepress-plugin-md-enhance

    强大且可树摇的 Markdown 增强插件。
    包括属性、标签页、代码标签页、提示框、脚注、标记、下标、上标、代码片段、图片、图片懒加载、图片缩放、图片标记、自定义对齐、任务列表、图表、流程图、公式、数学公式、图表 (mermaid)、幻灯片 (reveal.js)、代码演示等功能。

    @vuepress/plugin-shiki

    使用 Shiki 为 Markdown 代码块启用语法高亮

    @vuepress/plugin-prismjs

    使用 Prism.js 为 Markdown 代码块启用语法高亮

    @vuepress/plugin-search
